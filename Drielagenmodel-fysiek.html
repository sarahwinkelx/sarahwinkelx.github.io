Fysieke laag

De fysieke laag in een computer gaat over de hardware, zoals de processor. De processor (ook wel CPU genoemd) is het onderdeel dat alle berekeningen uitvoert en de belangrijkste taken van de computer regelt.

Kloksnelheid

Kloksnelheid is een belangrijke eigenschap van een processor. Dit geeft aan hoe snel de processor kan werken. Kloksnelheid wordt gemeten in gigahertz (GHz), wat betekent hoeveel miljard bewerkingen per seconde de processor kan doen. Een hogere kloksnelheid betekent vaak dat de computer sneller werkt, maar dit hangt ook af van andere factoren.

Verwerkingssnelheid

Verwerkingssnelheid is hoe snel de processor daadwerkelijk taken kan uitvoeren. Dit wordt niet alleen beïnvloed door de kloksnelheid, maar ook door het aantal kernen (cores) dat de processor heeft. Elke kern kan taken tegelijkertijd uitvoeren. Dus, hoe meer kernen, hoe beter de processor meerdere taken kan verwerken (multitasking). Andere factoren, zoals cache-geheugen en de efficiëntie van de processor, spelen ook een rol bij de totale verwerkingssnelheid.
